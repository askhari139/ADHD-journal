Core rules
Only 3 tasks per day
Written on paper, before screens. The constraint forces prioritization while your mind is still fresh — not mid-day when decisions are expensive.
~3.5 hours of deep work is enough
Intentional, not a bug. For ADHD, 3–4 hours of genuine focused work is what the brain can sustain before quality degrades. Most people spend 8 hours "working" and get 2 hours of real work done — scattered and exhausted. This makes those hours deliberate instead.
Exercise before deep work
Exercise raises dopamine and norepinephrine — the same pathway as stimulant medication — and the effect lasts 2–3 hours. It is one of the most reliable non-pharmacological ADHD interventions. Even 20 minutes of brisk walking counts.
Name the task before the timer
"Work on PKPD" is almost impossible to initiate. "Simulate theophylline 1-CPT in rxode2 with Ka=0.4" is not. Specificity is what makes starting possible.
Name productive procrastination
When you drift into something that feels productive but isn't your task — say it out loud: "I am choosing to reorganize files instead of my task." Converting unconscious drift into a decision is what makes it redirectable.
Systems fail at transitions, not tasks
You probably do fine inside a block. It's the gaps between blocks that break things. Know what you're doing next before you stop what you're doing now — every time.
Flat state is not a willpower problem
Change your environment first: stand up, go outside for 5 min, change rooms. State is physical. Then name the smallest possible version of the task — not "write code," but "open the file."

What happened?
⌛
Zoned out — lost track of time
Don't reconstruct the lost time. Write the time right now. Name your next action in one sentence. Set a 5-minute timer. Do only that.
🔀
Fell into productive procrastination
Close whatever you drifted into — don't finish it. Put it on the parking lot. Set a 2-minute timer and make contact with your actual task, even just reading what you last wrote.
🫥
Flat — can't start anything
State problem, not willpower. Stand up, go outside for 5 minutes. Come back and name the smallest possible version of the task. "Open the file" counts.
🔥
Hyperfocused on the wrong thing
Set an alarm for 10 minutes labeled with your actual task. When it fires, finish the sentence and switch. Redirect hyperfocus — don't fight it.
💀
Whole morning gone, nothing done
Don't write the day off. Run one 45-minute block right now on one task. One block done is not a failed day. The close-out still matters.
🙃
The system feels too rigid today
Fine. Minimum viable day: name one task, set one timer, do it. One block done beats zero blocks with a perfect system you didn't start.
